## Deployment

To deploy this project run

```bash
  yarn deploy
```


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`GOOGLE_ID`
`GOOGLE_CLIENT_SECRET`
`MONGODB_URI`
`NEXTAUTH_URL`
`NEXTAUTH_URL_INTERNAL`
`NEXTAUTH_SECRET`
